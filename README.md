# VIT Scheduler – Classroom Allocation System

An Android application developed to automate classroom allocation in educational institutions.

## Features
- Automatic classroom allocation
- Conflict-free scheduling
- Excel file upload support
- Room allocation based on:
  - availability
  - time slots
- Offline storage using Room Database
- Export allocation results to Excel

## Technologies Used
- Kotlin
- Android Studio
- Room Database
- Apache POI
- SQLite

## Advantages
- Reduces manual effort
- Prevents double booking
- Improves resource utilization
- User-friendly interface
